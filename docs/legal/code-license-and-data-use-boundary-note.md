# Code License And Data Use Boundary Note

## 1. Назначение

Документ фиксирует проектное допущение о том, что лицензионный режим программного кода не равен согласию на использование данных Заказчика. Эта предварительная policy note требует юридической проверки и не применяется без договорного основания.

## 2. Граница кода и данных

Лицензия на программный код, возможность локального запуска, модульная архитектура, open repository workflow или future commercialization не создают согласие Заказчика на передачу, обезличивание, агрегацию, обучение моделей или коммерческое использование данных.

## 3. Default-off flags

```text
telemetry_enabled=false
dataset_contribution_enabled=false
external_upload_enabled=false
ai_training_allowed=false
commercial_use_allowed=false
```

Эти значения являются проектным default-off допущением. Их изменение возможно только после отдельного согласия, договора или иного применимого правового основания.

## 4. Практическое следствие

Любой future data contribution, AI training signal, external upload или commercial use должен проектироваться как отдельный, проверяемый и обратимый контур. Он не должен включаться скрыто через лицензию, deployment profile, configuration default или acceptance report.
