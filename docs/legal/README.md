# Legal/Data Boundary Notes

`docs/legal/` содержит внутренние проектные policy notes для разграничения прав на код, customer-specific артефакты, данные Заказчика, возможный вклад обезличенных данных и правила агрегирования.

Эти документы не являются юридическим заключением, договором, пользовательским соглашением, политикой обработки персональных данных или согласием Заказчика. Они фиксируют проектные допущения для дальнейшей юридической проверки и не применяются во внешней договорной работе без отдельного утверждения.

## Артефакты

- `ip-and-deliverables-policy-note.md` — предварительная policy note о границе между программным кодом, универсальными компонентами и customer-specific документами.
- `customer-data-boundary-policy-note.md` — предварительная policy note о локальном контуре данных Заказчика и уровнях безопасной обработки.
- `data-contribution-policy-note.md` — предварительная policy note о default-off модели передачи, обезличивания, агрегации и использования данных.
- `anonymization-and-aggregation-policy-note.md` — предварительная policy note о pipeline обезличивания, обобщения, агрегирования и проверки риска повторной идентификации.
- `code-license-and-data-use-boundary-note.md` — предварительная policy note о том, что лицензионный режим программного кода не равен согласию на использование данных Заказчика.

## Проверка
