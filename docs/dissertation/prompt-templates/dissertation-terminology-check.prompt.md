# Dissertation Terminology Check Prompt

Проверь терминологию фрагмента диссертации.

Минимально проверить:

- расчетная единица;
- формирование локального сметного расчета;
- фактические данные;
- сопоставление;
- таблица соответствий;
- план-фактное сопоставление;
- контрольное решение по результатам план-фактного сопоставления.

Результат: список терминологических замечаний и предлагаемых замен без DOCX update.
