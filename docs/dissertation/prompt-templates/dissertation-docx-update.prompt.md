# Dissertation DOCX Update Prompt

Использовать только после принятого markdown patch.

Обязательные условия:

- DOCX update requires explicit user request.
- Render/visual check required after DOCX update.
- Do not alter bibliography numbering unless explicitly requested.
- Do not create a new DOCX version without explicit user request.

Результат должен перечислить accepted patch, целевой раздел, ручные шаги проверки и ожидаемый rendered review.
