# Dissertation Section Update Prompt

Подготовь markdown patch для указанного раздела диссертации.

Ограничения:

- не редактировать DOCX напрямую;
- не усиливать научные выводы;
- не добавлять источники без citation request;
- не писать, что Codex/GRACE является научным результатом;
- не писать, что Tartip автоматически выбирает ГЭСН.

Результат должен быть markdown patch, а не новая версия DOCX.
