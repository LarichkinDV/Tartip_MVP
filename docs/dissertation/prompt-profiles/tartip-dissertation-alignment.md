# Tartip And Dissertation Alignment

Диссертация описывает научную методику.

Tartip реализует прикладной проверяемый контур для локальной разработки, проверки и трассировки методики.

GRACE, Codex, execution packets, acceptance dashboards, verification dashboards и monthly defense dashboards являются инструментами разработки. Они не являются научным результатом диссертации.

Расчетная единица в диссертационном тексте может быть методическим связующим объектом расчетной цепочки.

В программной архитектуре Tartip `CalculationUnit` является нормативно-расчетным шаблоном.

Фактические данные в Tartip связываются преимущественно через `WorkPackage` / захватку / производственный пакет.

Запрещено механически переносить техническую архитектуру Tartip в научный текст.

Запрещено переносить старую диссертационную формулировку расчетной единицы как универсального контейнера в кодовую модель Tartip.
